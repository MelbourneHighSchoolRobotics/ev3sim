Compass Sensor¶

class ev3sim.devices.compass.ev3.CompassSensor(parent, relativePos, relativeRot)¶

EV3 Compass Sensor, calculates the bearing of the device relative to some direction (which can be specified).

To get this bearing, use value.

To set the relative heading of the sensor, use calibrate.

calibrate()¶

Set the sensor so the current bearing is interpreted as 0.

Example usage: ` >>> compass.calibrate() >>> # Rotates 10 degrees to the left >>> compass.value() 10 `

value()¶

Get the compass value.

Returns

Value from 0 to 360, which is the relative bearing (measured anticlockwise from the true bearing).

Example usage: ` >>> print(compass.value()) 48.2 `
